# Ticket Management System

A full-stack Ticket Management System developed using ASP.NET Core Web API, Angular and MySQL.

## Technologies

- ASP.NET Core 10
- Entity Framework Core
- MySQL
- Angular 22
- REST API
- Git

## Features

### Users
- List Users
- Create User
- Update User
- Delete User

### Tickets
- List Tickets
- Create Ticket
- Update Ticket
- Delete Ticket

## Project Structure

TicketApi/
ASP.NET Core Web API

ticket-app/
Angular Frontend
